# Belt Loop Images

This directory contains images of Cub Scout adventure belt loops from ScoutShop.org.

## File Naming Convention

Images should be named using the ScoutShop SKU number:
- Format: `[SKU].jpg` or `[SKU].png`
- Example: `614334.jpg` for a specific belt loop

## Organization

You can organize images in two ways:

### Option 1: Flat Structure (Recommended)
```
belt-loops/
  ├── 614334.jpg
  ├── 614335.jpg
  ├── 614336.jpg
  └── ...
```

### Option 2: By Rank (Alternative)
```
belt-loops/
  ├── tiger/
  │   ├── 614334.jpg
  │   └── ...
  ├── wolf/
  │   └── ...
  └── bear/
      └── ...
```

## How to Add Images

1. Download belt loop images from ScoutShop.org
2. Rename the file to match the SKU (found in the product URL or page)
3. Save as JPG or PNG format
4. Place in this directory

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
